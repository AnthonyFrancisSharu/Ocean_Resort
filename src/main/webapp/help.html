Help & Guidelines

Instructions for operating the Ocean View Resort Reservation System.

Getting Started

Welcome to the Ocean View Resort Reservation Management System. This system is designed to help front desk staff manage room reservations, guest records, and billing efficiently.

Use the navigation menu on the Dashboard to access all features. Only authorized staff members with valid login credentials can access this system.

💡 Default Login Credentials (Change after first login) Username: admin  |  Password: admin123

How to Add a New Reservation

  • Click "Add New Reservation" from the Dashboard.
  • The system will automatically generate a unique Reservation ID.
  • Fill in guest details: Full Name, Address, Contact Number, and Email (required).
  • Select the Room Type and enter the Room Number.
  • Choose Check-in and Check-out dates. Cost is auto-calculated.
  • Add any special requests from the guest if needed.
  • Click "Save Reservation" and note the Reservation ID for the guest.

How to View & Search Reservations

📋 Viewing All Reservations

  • Click "View & Search Reservations" from the Dashboard.
  • All reservations are listed in a table with ID, Guest Name, Contact, Room Type, Room No., Check-in, Check-out, and Status.
  • Use the Room Type filter dropdown to show only a specific room category (Standard, Deluxe, Suite, Presidential).
  • Use the Status filter dropdown to show only Active, Checked Out, or Pending reservations.
  • Results are paginated — use the page buttons at the bottom to navigate through records.
  • Click the View button on any row to open the full reservation details.
  • Click the Bill button on any row to go directly to the billing page for that reservation.
  • Click the Delete button to permanently remove a reservation (a confirmation prompt will appear).
📌 Status Colours Active — Guest is currently checked in  |  Checked Out — Stay completed  |  Pending — Upcoming arrival

🔍 Searching a Specific Reservation

  • Use the search bar at the top of the reservations table to find a record instantly.
  • Search by Guest Name — type part of the name and results filter in real time.
  • Search by Reservation ID — e.g. type RES-001 to find an exact booking.
  • Search by Contact Number — enter the guest phone number to locate their booking.
  • Once the result appears, click View to see full details including address, NIC, booking summary, and special requests.
  • From the detail view, click Generate Bill to proceed to billing for that reservation.

How to Generate & Print a Bill

  • Click "Calculate & Print Bill" from the Dashboard.
  • Enter the Reservation ID and click "Generate Bill".
  • System calculates room charges, 10% service charge, and 8% VAT.
  • Click "Print Bill" to print using the browser print dialog.
  • Click "Mark as Paid" after receiving payment from the guest.
📋 Tax Breakdown Room Charge + 10% Service Charge + 8% VAT = Total Due
